Unit 6 | Creating Digital Graphics

For unit 6, I had to create 2 digital graphics with one being a vector graphic and the other being a bitmap graphic. To create these graphics, I used paint to create the bitmap graphic and I used photoshop to create the vector graphic. For the bitmap graphic, I created an advert used to promote a new gaming console and for the Vector Graphic, I created a User Guide for the new gaming console so viewers would know how it works.
